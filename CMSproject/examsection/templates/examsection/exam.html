Help Instructions

  1. Submit the result in the following Excel format:
    • 1. First row, first column should be named 'Rollno' and its column should contain student's roll number.
    • 2. First row, second column should be named 'name' and its column should contain student's full name.
    • 3. From the first row's third column onwards, write subjects' names and students' respective marks in the columns.
    • 4. For a new subject entry, add it in the first row, next column.
    • 5. Once data has been submitted for a certain selected filter, you cannot add the result for it but you can edit it using view result.
